There are a number of client libraries available to help sending mail using the Postal platform.

* [Ruby](https://github.com/atech/postal-ruby)
* [Rails](https://github.com/atech/postal-rails)
* [Ruby (mail gem)](https://github.com/atech/postal-mailgem)
* [PHP](https://github.com/atech/postal-php)
* [Node](https://github.com/atech/postal-node)
* [Java](https://github.com/matthewmgamble/postal-java)
